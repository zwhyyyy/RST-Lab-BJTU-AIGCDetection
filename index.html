Introduction

With the rapid development of generative models, AIGC detection has become an important research topic for media forensics, platform governance, and trustworthy AI. In real online environments, detection models must handle diverse generation pipelines, image qualities, compression settings, and open-world distribution shifts. This project introduces our studies on method design for AIGC detection. Each paper entry highlights paper titles, paper links, open-source code, and concise summaries of each method, with a focus on open-world detection, deepfake detection, robust representation learning, and online social networks.

ODDN: Addressing Unpaired Data Challenges in Open-World Deepfake Detection on Online Social Networks

AAAI 2025 / AIGC detection / Open-world deepfake detection / Online social networks

Paper PDF Code ODDN

Despite significant advances in deepfake detection, varying image quality caused by different compressions on online social networks remains challenging. Existing methods often rely on paired raw and compressed images, while open-world scenarios contain many more unpaired compressed samples than corresponding raw versions. ODDN addresses this problem with open-world data aggregation (ODA) and compression-discard gradient correction (CGC), aggregating paired and unpaired correlations while reducing sensitivity to compression variations. Experiments on 17 popular deepfake datasets show that ODDN outperforms state-of-the-art baselines.

Unlocking the Potential of Lightweight Quantized Models for Deepfake Detection

IJCAI 2025 / Lightweight deepfake detection / Model quantization / Edge deployment

Paper Code QMDD

Deepfake detection is increasingly crucial due to the rapid rise of AI-generated content. Existing methods achieve high performance by relying on computationally intensive large models, making real-time detection on resource-constrained edge devices challenging. This work proposes a low-bit quantization framework for lightweight and efficient deepfake detection. The Connected Quantized Block extracts common forgery features via the quantized path while retaining method-specific textures through shortcut connections, and the Shifted Logarithmic Redistribution Quantizer mitigates near-zero information loss by unfolding unbalanced activations. Comprehensive experiments show 10.8x lower computational costs and 12.4x lower storage requirements while maintaining strong detection performance, even surpassing SOTA methods using less than 5% FLOPs.

SAGNet: Decoupling Semantic-Agnostic Artifacts From Limited Training Data for Robust Generalization in Deepfake Detection

IEEE TIFS / Limited-data deepfake detection / Semantic-agnostic artifacts / Robust generalization

Paper Code SAGNet

Deepfake detection becomes especially challenging when training data covers only limited semantic categories. Existing methods can show strong performance bias between seen and unseen classes, limiting real-world generalization. SAGNet decouples semantic-agnostic artifacts from content-specific distributions so the model focuses on universal authenticity cues rather than semantic content. This strategy improves generalization under limited training data, achieving performance comparable to models trained with 10 times more data and extending across different semantic categories as well as GAN-based and diffusion-based generation methods.

Leveraging Failed Samples: A Few-Shot and Training-Free Framework for Generalized Deepfake Detection

AAAI 2026 / Few-shot deepfake detection / Training-free framework / Generalization

Paper PDF Code FTNet

Recent deepfake detection studies often treat unseen sample detection as a zero-shot task, but real-world failed samples may be available for analysis. FTNet reframes this setting as few-shot deepfake detection and proposes a simple training-free framework that uses only one fake sample from the evaluation set, without parameter updates. During evaluation, each test sample is compared with known fake and real samples and classified by the nearest category. Experiments on AI-generated images from 29 generative models achieve new SOTA performance, with an average improvement of 8.7% over existing methods.

Leveraging Unlabeled Data from Unknown Sources via Dual-Path Guidance for Deepfake Face Detection

CVPR 2026 / Unlabeled deepfake face data / Dual-path guidance / Generalization

Paper PDF Code DPGNet

Existing deepfake detection methods rely heavily on static labeled datasets, while real-world scenarios contain large amounts of unlabeled fake face data from unknown sources. DPGNet introduces dual-path guidance to bridge domain differences across generative models and utilize unlabeled samples. It combines text-guided cross-domain alignment with curriculum-driven pseudo-label generation, forming a domain-invariant feature space and dynamically incorporating unlabeled data. Experiments on multiple mainstream datasets show strong improvements over existing techniques.
